Ouroboros

The Ouroboros is a machine built around a filter, a PT2399 delay, a comparator and a clock divider, all combined in a feedback loop. It's designed to work in a eurorack system, but works well a stand-alone noise box.

The entire project is fully open source and under a creative commons license so people can build, redesign, and tweak their own versions if they want.

Build Docs

All the information you need can be found in the Github Repository, but if you're just building an Ouroboros then the easiest thing is to Download the zip file from here, unzip it, and then look in the build folder.

Revisions

When getting the documentation, it's important to check which revision you've got. This should be clearly marked on the PCB. Make sure to then to into the build folder for that revision.

You'll find the following :-

  • bom.html lists all the parts in a nice webpage so you can check them off to make sure you've got everything.
  • build.md is a text file with useful build information.
  • ibom.html is an interactive guide showing you where to place each part.

There's some other files in there which probably need to be tidied up, but may be useful if you're trying to order all the parts yourself.
